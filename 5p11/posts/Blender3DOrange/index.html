Simple 3D Orange Tutorial

Tutorial on how to make a 3D orange in Blender

Modelling

  1. Delete the default cube: Select cube > X > Delete.
  2. Add a UV Sphere: Shift-A > Mesh > UV Sphere. Leave segments/rings default.
  3. Shade Smooth: With sphere selected in Object Mode, Right-click > Shade Smooth.
  4. Enter Edit Mode: Tab.
  5. Enable Proportional Editing: press O (toggle on).
  6. Select the top vertex: use Vertex select (1 or the toolbar), click the top vertex.
  7. Move it down slightly to create a small recess: press G then Z, drag down while mouse wheel adjusts influence radius; make a shallow recess.
  8. Add a Subdivision Surface modifier: Properties > Modifiers tab (wrench icon) > Add Modifier > Subdivision Surface. Keep Levels View and Render at the defaults.
  9. Add a simple stem: Shift-A > Mesh > Cylinder. In the Add dialog (bottom-left) Move the cylinder into the recess (G), then scale in Edit Mode: select top face > S to scale down.
  10. Position and parent: position stem so it sits in the recess; optionally Shade Smooth the stem and join to the orange visually (Object > Set Origin > Origin to Geometry if needed).

Texturing

  1. Open the Shading workspace (top tabs) and select the sphere object.
  2. Click New to create a material.
  3. Base color: set Principled BSDF Base Color to hex #ED8800 (a good orange color).
  4. Node additions and wiring (Shift-A to add nodes):
    • Add Voronoi Texture
    • Add 2 ColorRamp
    • Add Displacement
    • Connect: Voronoi Texture Distance to both -> Color Ramp Factor
    • Connect: one Color Ramp Color -> “Roughness” Color Ramp` Color -> “Height”
    • Connect: Displacement Height -> Material Output Displacement

  5. Stem material: Click on the cylinder (while in object mode) and then create a second material for the cylinder. Use a green base (e.g., #3F9300FF).

Quick export

  • File > Save As to save your .blend.
  • Render > Render Image (F12). When finished: Image > Save As to export PNG.
