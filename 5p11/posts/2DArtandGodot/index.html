Simple 2D Background Art in LibreSprite and Using What We've Made in Godot

Tutorial that teaches you how to make 2D background assets, and import 2D and 3D assets into Godot

Simple 2D Background Art in LibreSprite and Using What We’ve Made in Godot

This tutorial walks through a beginner-friendly workflow:

  1. Create simple 2D background asset and platform in LibreSprite
  2. Export them as PNG files
  3. Import and place them in a Godot 2D scene
  4. Build a simple 2D scene
  5. Import and place a 3D asset in a 2D scene

Prerequisites

  • LibreSprite installed
  • Godot 4.x installed
  • The 3D Orange from last tutorial

Optional: graphics tablet, but mouse works fine.

Plan Your Background Assets

Before drawing, decide what layers your scene needs. A simple example:

  • Background Sky (solid color or gradient)
  • Mid tree
  • Foreground Platform

Recommended asset sizes

Pick a target game resolution first (example: 1280x720). Your asset resolution will follow from that decision. Your sky background will want to be the full size of a screen. A mid ground tree might be 256px or 512px tall.

Example folder plan:

  • assets/backgrounds/sky_back.png
  • assets/backgrounds/trees_mid.png
  • assets/backgrounds/foreground_plat.png

Create Assets in LibreSprite

  1. Create a new canvas

In LibreSprite:

  • File -> New
  • Set width/height (example 1280 x 720)
  • Background: transparent (for non-sky layers)

For pixel art, use integer scaling and keep brush sizes small (1–4 px).

  1. Set up palette and style

Keep color count small and readable:

  • 1–2 sky colors
  • 2–3 tree/ground colors

This keeps backgrounds cohesive and fast to make.

  1. Draw by layer depth (far to near)

Create separate LibreSprite files (or layers) for:

  • sky
  • mid
  • near
  1. Export PNG files

For each layer:

  • File -> Export
  • Format: PNG
  • Keep transparency for non-sky layers
  • Use clear names, e.g.:
    • sky_back.png
    • trees_mid.png
    • foreground_plat.png

Import 2D Assets into Godot

  1. Copy files into project folder

Place exported PNGs into your Godot project, for example:

res://assets/

Godot auto-imports assets when files appear in the project.

Build the Background for a Scene in Godot

Create a 2D background scene by creating a sprite for each of the 2D assets we made: Example hierarchy:

  • Sprite2D(sky_back.png)
    • Sprite2D(foreground_plat.png)
    • Sprite2D(trees_mid.png)

Adding the 3D Orange

  1. Create the 3D render viewport
  • Create new scene
  • Click + (Add Child Node).
  • Add SubViewport. Rename it to OrangeViewport.
  • With OrangeViewport selected, in Inspector:
    • Set Size = 512, 512 (or your preferred size)
    • Set Transparent Bg = On
    • Ensure Disable 3D = Off (must allow 3D)
  1. Build the 3D mini-scene inside the viewport
    • Select OrangeViewport → Add Child Node → Node3D. Rename to OrangeWorld.
    • Select OrangeWorld → add Camera3D.
    • Select OrangeWorld → add DirectionalLight3D.
    • Add your orange model:
    • Select the .glb/.gltf of the Orange from last tutorial: drag it from FileSystem onto OrangeWorld.
  2. Position camera and light
    • Select Camera3D:
    • Set Position to something like (0, 0, 3).
    • Enable Current = On. - Select DirectionalLight3D:
    • Rotate it so the model is lit (example X -45, Y 30, Z 0).

  1. Save 3D Scene/
    • CTRL + S to save 3D render port. Rename it to 3D_Orange.tscn.
  2. Display viewport output in 2D Scene
    • Go to previous 2D scene where we set up out background sprites
    • Select Background sprite → Add Child Node → Sprite2D. Rename to OrangeSprite.
    • Add the orangeViewport scene as a child of OrangeSprite by dragging 3D_Orange.tscn onto OrangeSprite
    • Resize for the Scene
    • Select OrangeSprite, in Inspector:
    • Texture → click dropdown → New ViewportTexture
    • Click the new ViewportTexture resource to edit it
    • Set Viewport Path (or Viewport Path in Scene) to OrangeViewport - Move OrangeSprite to where you want it in the 2D scene.
  3. Run and test
    • Press F6 (run current scene).
    • Confirm 3D orange appears in 2D Scene.
